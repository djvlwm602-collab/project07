# CRIT. — Portfolio Review by Industry Pros

당신의 포트폴리오를, 전문가의 책상 위로. 현업 PO·디자인 리드가 당신의 작업을 들여다봅니다. 리뷰어는 직접 고르세요. 크리틱은 5분이면 도착합니다.

## 핵심

- 포트폴리오 스크린샷 + 맥락 한 줄을 올리면, 선택한 리뷰어들이 각자의 관점으로 강점·우려·제안을 돌려줍니다.
- 리뷰어 풀은 확장 가능한 구조로, 현업 PO·디자인 리드 캐릭터를 계속 추가할 예정입니다.
- 본 서비스의 리뷰어는 가상의 캐릭터이며, 실제 인물·기업과 관련이 없습니다.

## 개발

```bash
npm install
npm run dev       # 개발 서버
npm run lint      # ESLint
npm test          # Vitest
npm run build     # 프로덕션 빌드
```

### 환경 변수

`.env.local`:

```
GEMINI_API_KEY=...
GEMINI_MODEL=gemini-2.5-flash-lite   # 선택, 기본값
```
